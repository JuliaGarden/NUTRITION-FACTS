# NUTRITION-FACTS
## Описание
#### nutritionist.py — это лёгкий и быстрый инструмент для планирования рациона, который помогает:
*  Рассчитать процент от дневной нормы нутриентов для каждого продукта
*  Подобрать топ-3 рецепта на основе введённых ингредиентов
*  Получить структурированный отчёт прямо в терминале
